Poly Image Plane - ZLOC Part 1

Sat, Dec 14, 2019

  • An Image Plane is just a “Reflection” of a camera’s Film Back.

  • We can figure out the Image Plane’s width and height by using some simple math.

Lets make a Poly Image Plane by using this formula.

  • First, create a camera. We will be using it’s default settings.

  • Focal Length(mm): 35
Horizontal Film Aperture(inch): 1.417
Vertical Film Aperture(inch): 0.945

  • Next, create an image plane, and set the depth to 35.

  • Now create a Poly Plane and rotate it 90 degrees in the X AXIS and FREEZE it.

  • And move the Poly Plane by -35 in the Z AXIS.

Finally here comes the fun part, Expressions!

  • Open the Expression Editor, copy and paste the code below, and hit the create button.

  • pPlane1.sx = cameraShape1.hfa * 2.54 * pPlane1.tz / (cameraShape1.fl / 10) * -1;
pPlane1.sy = cameraShape1.vfa * 2.54 * pPlane1.tz / (cameraShape1.fl / 10) * -1;

  • Viola! The Poly Plane matches the shape of the Image Plane. Play around with the Poly Plane’s TranslateZ attribute to see how it mimics the Image Plane’s Depth attribute.

  • And if you want the Poly Plane to follow the camera’s translations, just group the Poly Plane, and parent constraint the group to the camera.
