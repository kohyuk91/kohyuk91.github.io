Install Houdini Apprentice - CentOS7

Wed, Feb 19, 2020

Download Houdini

Install Houdini

  1. Open a terminal.
  2. Unpack the downloaded tar.gz archive. For example, 
    • $ tar -xvzf houdini-17.5.229-linux_x86_64_gcc6.3.tar.gz
  3. Run the houdini.install script: 
    • $ cd houdini-17.5.229-linux_x86_64_gcc6.3/
$ su
# ./houdini.install
    • Install details
  4. The licensing menu will appear during the installation process if no installed licenses can be detected
    • If you are installing Apprentice or Indie, choose Apprentice Licensing. This will take out the complexities of running a local license server.
  5. Error!!!
    • localhost.localdomain 어쩌구 저쩌구

애러 해결

  • Hostname이 localhost.localdomain 인것이 원인. Hostname을 유저 이름으로 바꿔주면 된다.
  1. 터미널 열기 
    • <유저이름 확인>
$ echo $USER
ex) hko

    • <Hostname 변경하기>
$ su
# hostnamectl set-hostname <새로운 호스트네임==유저이름>
ex) # hostnamectl set-hostname hko

    • <변경사항 확인>
$ su
# hostname
ex) hko

Houdini License Administrator

  1. Houdini License Administrator 실행
    1. 프로그램 > Side Effects Software » License Administrator…
  2. License Server 변경하기
    1. File » Change License Server
    2. License Server Host: <바뀐 Hostname> ex)hko
    3. Accept
  3. 라이센스 키 생성하기
    1. SideFX 로그인(https://www.sidefx.com)
    2. Non-commercial License for Houdini 들어가기
      1. Please choose the non-commercial product: Houdini FX (Non-Commercial)
      2. Server name: <바뀐 Hostname> ex)hko
      3. Server code: <Houdini License Administrator에서 확인 / Server Information 탭>
      4. Generate License
  4. 라이센스 키 입력하기
    1. 생성된 키들 하나씩 복붙
      • SERVER …
      • LICENSE …
      • …

alias 설정

  1. init.env에 alias 추가 
    • alias houdini="cd /opt/hfs18.0 && source houdini_setup && houdini"

Houdini 실행

  1. 터미널 열기 
    • $ houdini
